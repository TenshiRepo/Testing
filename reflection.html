Reflection on My Portfolio

1. Which component did you find most flexible?

The most flexible component, in my opinion, was the Card component. The structure of various content types, including the image, title, text, quote, and button, was quite simple. Bootstrap's utility classes within the card body and the ability to combine different card elements (header, body, and footer) made it adaptable to various design needs.

2. What challenges did you face with layout or alignment?

At first, I faced some challenges with vertical alignment, especially in making sure the elements within the Card were properly aligned, particularly the image and the text. I also had a little trouble getting the footer to stay consistently at the bottom of the page with varying content height, but the fixed positioning and flexbox utilities helped. Using d-flex, align-items-center, and justify-content-center was key to solving these issues.

3. If you could improve this with custom styles or JS, what would you add?

If I could improve this with custom styles and JavaScript, this would be Custom Animations. I'd use CSS transitions and animations to make the page feel more dynamic. For example, I'd add subtle fade-in effects when the page loads, and hover effects to the buttons and social media icons.
